ActiveCampaign Integration — Coming Soon
Coming Soon
This module will automatically push enriched leads into ActiveCampaign, enroll them in vertical-specific sequences, and report outcomes back to Lead Booster. Configure your API credentials below when ready to activate.
📤 Lead Booster → ActiveCampaign
✓ Pushes every enriched contact with verified email (ICP 60+) to ActiveCampaign as a new contact
✓ Sets custom fields: ICP Score, ICP Rationale, Vertical, Company, Domain, Enrichment Score
✓ Applies tier tag automatically: Elite (90+), Strong (75–89), Qualified (60–74)
✓ Enrolls contact in the matching vertical sequence (Construction, Behavioral Health, Healthcare, Medical Transport)
✓ Marks contact source as Lead Booster Pro for attribution
📥 ActiveCampaign → Lead Booster
✓ Fires a webhook when a contact replies, books a meeting, or is marked as a deal
✓ Outcome is written back to lb_contacts in Supabase (outcome field)
✓ ICP scoring model learns from outcomes over time — closed deals improve future scoring
✓ Weekly performance report: open rates, reply rates, meetings booked per vertical and ICP tier
✓ Dashboard Intelligence card shows sequence performance alongside pipeline health
System Components
These workflows and tables will be built when integration is activated
NOT BUILT
WF-13 Contact Push
n8n Workflow
Triggers at end of WF-06. Pushes each contact with valid email and ICP 60+ to ActiveCampaign via API. Creates or updates contact, sets custom fields, applies tier tag, enrolls in sequence.
NOT BUILT
WF-14 Outcome Receiver
n8n Workflow
Webhook listener. Receives outcome events from ActiveCampaign automations (replied, meeting booked, deal closed). Matches to lb_contacts by email, writes outcome to lb_outcomes table.
NOT BUILT
WF-15 Performance Reporter
n8n Workflow
Weekly pull from ActiveCampaign API. Fetches open rates, reply rates, meetings booked per vertical and ICP tier. Stores in lb_ac_reports. Surfaces in dashboard Intelligence card.
NOT BUILT
lb_outcomes
Supabase Table
Stores outcome data per contact. Fields: contact_id, outcome (replied/meeting/qualified/closed/disqualified), outcome_date, was_icp_accurate, score_delta.
NOT BUILT
lb_ac_reports
Supabase Table
Stores weekly ActiveCampaign performance reports per vertical and ICP tier. Used by dashboard Intelligence card to show sequence ROI.
NOT BUILT
AC Sequences (x4)
ActiveCampaign
Four vertical-specific email sequences to be built in ActiveCampaign: Construction, Behavioral Health, Healthcare, Medical Transport. Each tailored to that vertical's pain points and ManageAI's value prop.
